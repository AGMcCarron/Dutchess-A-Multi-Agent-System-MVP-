**Folders**

Assistants: A folder where the Assistant ID's are stored
Conversations: A folder where past conversations are stored
Data Base: A folder where any data generated by the Resource Creator is stored
Documents: A folder where the knowledge base is stored
Output Images: A folder where the images taken from the knowledge base is stored

**Notebooks in order of which to run first**
1 - Dutch Resource Creator: Processes files in the Documents folder and outputs to the Output Images and Data Base folders
2 - Dutch Assistant Creator: Runs the assistant creation process and stores assistant ID's in the Assistants Folder
3 - Dutch Conversation Script: Where the conversation with the chatbot is held, outputs conversations to the conversations folder
